Changelog & Releases

Stay up to date with the latest features, improvements, and bug fixes for FeederMate.

v1.6.0 Latest

June 11, 2026

Added

  • Multi-Rod Support in Casual Sessions: Anglers can now track up to two rods independently in Casual mode, with independent tabs, separate cast timers, separate presets, and current distance tracking.
  • Rod Active Pulse Indicators: Selectable rod tabs feature pulsing active dots indicating if a cast timer is currently active on that rod.
  • Rod-Attributed Catch Log Badges: Displays localized R1 and R2 badges on catch rows within active session feeds to easily track which rod caught the fish.
  • Distance Preset Picker: Added a dedicated distance picker widget with quick-select preset chips and a custom distance input dialog.
  • Dynamic 3D Flip Timer: Tapping the countdown progress ring flips it with a 3D animation, revealing the elapsed session duration (FISHING TIME) for 3 seconds before automatically flipping back.
  • Hourly Color Progression: For sessions without an end time, the progress ring (accent and knob) darkens by 8% per hour elapsed (clamped at 15% minimum) to indicate 60-minute intervals on the bank.

Changed

  • Timer Track Contrast: Adjusted the circular timer background track opacity to 15% of the active accent color to enhance outdoors, direct-sun visibility.
  • Localization Updates: Fully localized all new multi-rod badges, tabs, alerts, and settings in English, Hungarian, and German.

v1.5.0

June 5, 2026

Added

  • Home Screen Performance Dashboard: When no session is active, the home screen now shows a stats dashboard with four cards — Total Caught Weight, Total Fish Count, Personal Best (with species name), and Podium Cabinet (Gold / Silver / Bronze counts with medal colours).
  • Last Outing Card: Displays a summary of the most recently completed session on the home screen, including date, session type, and key weight/catch stats.
  • Centered + Button Navigation: Redesigned the bottom navigation with a prominent centre + button, fixing a visual glitch that showed a black rectangle above the navigation bar on some Android devices.
  • Session Picker: Tapping + now shows a clean menu with three clearly labelled session options (Casual, Practice, Competition), each with a coloured icon and description.
Home Dashboard Screenshot Placeholder

Changed

  • Bottom Navigation Fix: Fixed a visual bug where a black rectangle appeared above the navigation bar on Samsung and other Android devices.
  • Dashboard Card Design: Redesigned all four dashboard tiles with consistent sizing, rounded corners, coloured icon badges, bold values, and clear labels — optimised for outdoor sunlight readability.
  • Podium Card: Podium finishes are now shown as individual Gold / Silver / Bronze rows with their respective metallic colours rather than icon counters.
  • Session Picker Display Fix: Fixed a layout issue where the session picker was cut off at the bottom on devices with on-screen navigation buttons.

v1.4.0

June 5, 2026

Official Rebranding

  • Renamed to "Feeder Mate": The app name has officially changed from FeederPulse to Feeder Mate across all platforms — Android, iOS, and all in-app text.
  • Standardized Branding Color Theme: Updated all primary, secondary, and supporting accent colors to the new official palette — Sage Green, Ocean Blue, Warm Amber, and a deep dark background.
  • Clean App Icon: Integrated the new official icon across Android and iOS launcher assets.

Added

  • Full-Screen Splash Screen: Added a smooth full-screen splash that transitions to the home page after launch, fixing scaling issues seen on various devices.
  • Android 12+ Splash Screen Fix: Fixed the launch screen on Android 12 and newer to show the full-screen background image instead of a small icon on a plain background.
  • iOS Launch Screen Fix: Fixed the iOS launch screen to display a clean, full-screen background image without any distortion or stretching.

Changed

  • Statistics Chart Alignment: Shifted telemetry line and bar chart colours from Sage Green (primary) to Ocean Blue (secondary), aligning with the new branding guidelines.

v1.3.1

June 2, 2026

Added

  • Drag-to-Delete Cast Timer Presets: Long-pressing any preset chip (including built-in defaults) lifts it into a vertical drag gesture, revealing a vibrant red "Drag here to delete" drop zone. Deletion is persisted to the database.
  • Horizontal Preset Auto-Scroll Centering: The selected active preset chip is automatically scroll-centered on selection or screen load.
  • Pencil Icon for Custom Presets: Swapped the text-based "Custom" chip button with a clean pencil icon, consistent across all session types.
  • Active Preset Protection: Long-tapping the currently selected (active) preset opens the duration refinement slider instead of triggering drag-to-delete, preventing accidental timer changes.

Changed

  • Preset Auto-Sorting: Newly added or adjusted cast timer presets are automatically sorted in ascending order for a clean, predictable layout.
  • Consistent In-App Notifications: All in-app notifications now look and behave the same — they dismiss automatically after 3 seconds and can be swiped away.

v1.3.0

May 30, 2026

Added

  • Session Pulse Analysis: A premium, rhythm-based curved line chart that visualises your session's rhythm, momentum, and performance trends in real time.
  • Rhythm & Pulse Metrics: Dynamic calculation of Peak/Average Pulse scores, standard deviation, and session stability.
  • Smart Period Detection: Automated, sliding 40-minute "Best Window" calculation and low-activity "Dead Zones" detection.
  • Automated Bank Insights: Textual summaries detailing catch pace, lulls, momentum spikes, and trailing fatigue.
  • History Pulse Sparklines: Mini visual pulse fingerprints embedded inside finished session list items.

Added

v1.1.3 (Build 5)

May 25, 2026

This milestone release introduces a fully customizable, drag-and-drop Active Session Layout for advanced tracking, alongside a beautifully simplified Casual (Everyday Leisure) Mode.

Key Enhancements

  • Customizable Active Session Layout (Beta): Take complete control of your bank-side active workspace with interactive, reorderable widgets.
  • Casual / Leisure Active Session Mode: Log leisure fishing sessions with a beautiful, green-themed interface completely stripped of competitive elements.
  • Integrated Dashboard & 3D Flip Timer: Reduced the circular timer frame and embedded it inside the presets control panel with a premium 3D flip interaction.
  • Smart Distance Presets & Premium Bottom Sheets: Seamless cache reusability and modernized dialog layouts.
  • Global Timer Mode: Added a new setting to toggle between Count Up and Countdown timer modes globally.
  • Practice End Times: Added support for an optional end time for Quick Practice sessions.

Fixed & Polished

  • Session End Notification Fix: Fixed an issue where ending a session early could leave cast timer notifications running in the background.
  • Background Opacity Fix: Corrected catch detail sheets to render with solid surface colors.
  • Distance Chart Resiliency: Fixed an issue in the Catch Distance Chart.
